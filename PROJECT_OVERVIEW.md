# Orbix - Project Overview

## What is Orbix?

**Orbix** is a unified productivity platform that combines 6 essential business tools into one seamless application. The tagline is **"Smart Solutions for Your Work"**.

## The 6 Tools

### 1. Freelance Invoice Generator
Create, manage, and track professional invoices with:
- Client management
- Invoice templates
- Payment tracking (Paid/Pending/Overdue)
- PDF generation
- Email sending
- Multiple currencies
- Tax/VAT support

**Target Users**: Freelancers, small businesses, agencies

### 2. AI Writing Assistant
AI-powered content creation and improvement:
- Blog posts, emails, social media content
- Grammar and spelling correction
- Text rewriting (different tones)
- Text summarization
- Local language support (Kenyan English, Sheng, Kiswahili)
- Document templates (CV, cover letters, proposals)
- AI chat mode

**Target Users**: Students, freelancers, content creators, businesses

### 3. Contract & Proposal Generator
Generate professional legal documents:
- Freelance contracts
- Employment agreements
- Partnership contracts
- Service agreements
- Business proposals
- Tender applications
- AI-assisted writing
- Template library
- Document storage

**Target Users**: Freelancers, SMEs, agencies, entrepreneurs

### 4. Time Tracking & Billing
Track work hours and generate bills:
- Start/stop timer
- Manual time entries
- Project tracking
- Client tracking
- Automatic billing calculation
- Invoice generation from time entries
- Reports and analytics
- Team support

**Target Users**: Freelancers, agencies, consultants, contractors

### 5. Resume & Portfolio Builder
Create professional resumes and portfolios:
- ATS-friendly templates
- Auto-formatting
- AI resume writer
- Job-specific tailoring
- Portfolio website builder (no-code)
- Multiple resume versions
- PDF/DOCX export
- Online portfolio links

**Target Users**: Job seekers, freelancers, students, professionals

### 6. AI-Powered Resume Optimizer
Optimize resumes for job applications:
- ATS keyword matching
- Resume scoring
- Job description analysis
- Improvement suggestions
- Missing keyword detection
- Achievement enhancement
- Cover letter generation
- LinkedIn profile optimization

**Target Users**: Job seekers, career changers, professionals

## Key Features

### Unified Platform
- Single sign-up for all tools
- Shared data between tools
- Consistent UI/UX
- Centralized dashboard

### Data Integration
- Time tracking → Invoice generation
- Client data shared across tools
- Resume builder → Resume optimizer
- Contract generator uses client data

### AI-Powered

- Intelligent optimization

- FastAPI AI service
- PostgreSQL + MongoDB + Redis
